# python-lab2
# HARD Task: Telegram Bot (menu / scream / whisper)

Бот створений на основі TutorialBot.py.

## Інструкція

1. Клонувати репозиторій
2. Створити .env з:
   BOT_TOKEN=your_telegram_token_here
3. Встановити залежності:
   pip install -r requirements.txt
4. Запустити:
   python bot.py

## Команди

/start – почати
/menu – показати список команд
/scream <text> – написати ВЕЛИКИМИ літерами
/whisper <user_id> <text> – приватне повідомлення
